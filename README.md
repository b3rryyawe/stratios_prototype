# STRATIOS PROTOTYPE — Strategic Decision Engine

STRATIOS is an interactive strategic simulation modelling the impact of geopolitical lithium supply shocks on a global EV manufacturer.

## Scenario
You play as CEO navigating:
- Supply chain disruption
- Input cost inflation
- Dependency risk
- Structural strategy trade-offs

## Features
- Dynamic shock modelling
- Multi-stage strategic decision tree
- Non-linear strategic scoring system
- Risk vs resilience assessment
- Interactive Streamlit interface

## Built With
- Microsoft Excel
- Python
- Streamlit

## Run Locally

pip install -r requirements.txt
streamlit run app.py
